# okky-clone-backend
OKKY 개발자 커뮤니티 백엔드 (Spring Boot + MySQL)

## Features
### 인증/계정
- 세션 기반 로그인/로그아웃

### 게시판/게시글
- Topic 별 게시판 분리
- 게시글 목록/상세 조회(비회원 포함)
- 게시글 작성/수정/삭제(회원)

### 댓글
- 댓글 작성/수정/삭제(회원)
- 답글 지원 (Depth 1 제한, 부모 댓글에 여러 답글 가능)

### 추천, 비추천 기능
- 게시글 추천, 비추천 가능(회원)
- 추천, 비추천 취소 가능

## Tech Stack
Java, Spring Boot, MySQL, Redis(예정), AWS(예정)
